# **日內邊界反應交易計劃 (15K → 5K → Pin Bar)**

## **一、交易哲學**

1. **分工明確**

   * **15K** → 決定報酬空間 (R 倍數)
   * **5K** → 決定市場是否配合 (勝率)
   * **Pin Bar / K 棒反應** → 具體進場與停損位置

2. **系統優先**

   * 停損位置永遠基於結構，而非主觀喜好
   * 不追勝率、不追成交量噪音
   * 勝率低可接受，但每筆交易須有明確風險報酬

---

## **二、交易刻度與條件**

### **1. 15K：決定 R 倍數與邊界**

* **邊界判定**：W 底、破底回拉原邊界、突破 W 頭回測
* **R 倍數**：

  * 計算假設進場後，價格到邊界上沿可達 R ≥ 3
  * 若 R < 3 → **禁止交易**
* **結構評估**：

  * 僅在明確邊界存在時進入下一步
  * 邊界尚未形成 → **不做**

---

### **2. 5K：決定市場配合與進場訊號**

#### **5K 結構邊界定義**

* **5K 結構邊界** = 支撐/阻力區間,由 5K 圖上多次觸及的價格範圍形成
* **邊界下沿** = 支撐區最低點
* **邊界上沿** = 阻力區最高點
* **邊界內** = 價格在支撐與阻力之間的區間

#### **允許進場型態**

1. **W 底邊界反彈**
2. **破底又拉回原邊界**
3. **突破 W 頭後回測邊界**

#### **評估條件(決定是否往下進入 Pin Bar 判斷)**

* 出現上述任一型態 → **通過,進入 Pin Bar 判斷**
* 未出現任何型態 → **停止交易**
* 型態不明確或邊界模糊 → **停止交易**

---

### **3. Pin Bar 品質判斷**

#### **Pin Bar 進場區定義**

* **Pin Bar 進場區** = Pin Bar 本身形成的價格區間
* **進場區下沿** = Pin Bar 下影線低點(停損位置)
* **進場區上沿** = Pin Bar 收盤價附近(進場參考)
* **進場區內** = Pin Bar 的 K 棒範圍

#### **品質標準（篩選條件）**

| 條件     | 可進場標準                 | 不可進場標準                   |
| ------ | --------------------- | ------------------------ |
| 下影線    | ≥ 實體 2 倍              | < 實體 2 倍                 |
| 上影線    | < 下影線 50%            | ≥ 下影線 50%（上影線過長）         |
| 實體     | 收盤價 ≥ K 棒高度 70%      | 收盤價 < K 棒高度 70%（收盤過低信號弱） |
| 停損合法性  | 守 pin bar 低點 → R 倍數 ≥ 3 | R 倍數 < 3 或停損超過合理範圍       |
| 交易次數限制 | 同一 Pin Bar 使用次數 < 2   | 同一 Pin Bar 已使用 ≥ 2 次     |

#### **進場方式**

* **進場時機**: 隔壁 K 棒收盤價離開 Pin Bar 進場區後進場
* **停損位置**: Pin Bar 下影線低點
* **訊號失效**: 隔壁 K 棒收盤價未離開 Pin Bar 進場區 → **禁止交易**

---

## **三、風控規則（不交易情況）**

1. **R 倍數不足**

   * 15K 評估後，R < 3 → 不做
2. **邊界已損兩次**

   * 同一 15K 邊界已被停損兩次 → 不再交易該邊界
3. **單日累計損失次數**

   * 當日已損 5 次 → 停止交易
4. **Pin Bar 過長下影線**

   * 無法守停損 → 禁止交易
5. **訊號失效**

   * 隔壁 K 棒收盤價未離開 Pin Bar 進場區 → 不交易

---

## **四、停損與目標**

* 停損：**固定於 pin bar 低點**
* 目標：邊界上沿
* 勝率低可接受，只要價格達到目標或停損明確
